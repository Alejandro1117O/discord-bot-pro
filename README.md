# 🤖 Discord Bot PRO v3.0

**Bot Discord Profesional con Moderación Avanzada, Sistema de Niveles y Utilidades**

---

## ✨ Features

### 🛡️ Sistema de Moderación Completo
- **Ban/Kick/Mute** - Controla a los usuarios infractores
- **Sistema de Advertencias** - Auto-ban después de 3 advertencias
- **Logs Automáticos** - Registro de todas las acciones
- **Anti-spam** - Protección automática contra spam

### 🎫 Sistema de Tickets
- Crear canales de soporte privados automáticamente
- Gestión de permisos por ticket
- Historial guardado

### ⭐ Sistema de Niveles/XP
- Ganancia de XP por mensajes
- Level up automático con notificaciones
- Top 10 usuarios (tabla de clasificación)

### 📊 Utilidades
- **Encuestas interactivas** - Con reacciones
- **Sorteos/Giveaways** - Premios automáticos
- **Lock/Unlock canales** - Control de acceso
- **Gestión de roles** - Crear/eliminar roles
- **Estadísticas del servidor** - Información en tiempo real

## 📦 Comandos Principales

### Moderación
```
!ban @usuario <razón>          - Banear usuario
!kick @usuario <razón>         - Expulsar usuario
!mute @usuario <tiempo>        - Silenciar usuario
!warn @usuario <razón>         - Advertir usuario
!advertencias @usuario         - Ver advertencias
```

### Información
```
!info                           - Info del servidor
!usuario @usuario              - Info de usuario
!stats                         - Estadísticas
!ping                          - Latencia del bot
```

### Tickets
```
!ticket crear                  - Crear nuevo ticket
!ticket cerrar                 - Cerrar ticket
!ticket agregar @usuario       - Agregar usuario
!ticket remover @usuario       - Remover usuario
```

### Niveles
```
!nivel [@usuario]              - Ver tu nivel
!top                           - Top 10 usuarios
!xp                            - Ver tu XP
```

### Utilidad
```
!encuesta <pregunta>           - Crear encuesta
!giveaway <premio>             - Crear sorteo
!lock                          - Bloquear canal
!unlock                        - Desbloquear canal
!limpiar <número>              - Limpiar mensajes
```

## 🚀 Instalación Rápida

### 1. Clonar el repositorio
```bash
git clone https://github.com/Alejandro1117O/discord-bot-pro.git
cd discord-bot-pro
```

### 2. Instalar dependencias
```bash
pip install discord.py python-dotenv
```

### 3. Configurar token
```bash
cp .env.example .env
nano .env
```

Agrega tu token de Discord:
```
DISCORD_TOKEN=tu_token_aqui
```

### 4. Ejecutar el bot
```bash
python3 bot_v3_profesional.py
```
